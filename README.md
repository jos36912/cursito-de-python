# Cursito de Python

Repositorio personal de estudio para aprender Python desde cero, siguiendo un enfoque progresivo y práctico.

El objetivo de este repositorio es consolidar fundamentos del lenguaje mediante ejercicios simples y código claro, priorizando la comprensión sobre la complejidad innecesaria.

## Contenido

El repositorio está organizado por temas:

- `variables/`  
  Uso básico de variables y asignación de valores.

- `tipos_de_datos/`  
  Tipos de datos simples y compuestos.

- `operadores/`  
  Operadores aritméticos y de comparación.

- `condicionales/`  
  Estructuras `if`, `elif` y `else`.

- `ejercicios/`
  Scripts interactivos para poner en práctica la lógica de programación:
  - `juego.py`: Implementación del juego "Piedra, Papel o Tijera". Utiliza un bucle `while` infinito para mantener el juego activo, el módulo `random` para la elección del bot y un diccionario para validar las reglas de victoria de forma eficiente.
  - `mini_bot.py`: Bot conversacional que solicita datos al usuario y valida el tipo de entrada (letras vs números) usando métodos de strings.
  - `valida_edad.py`: Ejercicio introductorio de condicionales para verificar la mayoría de edad.

Cada carpeta contiene ejemplos y pequeños ejercicios relacionados con el tema.
