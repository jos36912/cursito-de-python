# Curso de Python

Este repositorio contiene una serie de ejercicios y ejemplos para un curso de Python.

## Contenido

El proyecto está estructurado en las siguientes carpetas y archivos:

-   `holamundo.py`: Un simple programa "Hola Mundo" en Python.
-   `OPERADOR ARITMETICO.txt`: Un archivo de texto que describe los operadores aritméticos.
-   `tipos de datos/`: Carpeta que contiene ejemplos de tipos de datos en Python.
    -   `datos-simples.py`: Ejemplos de tipos de datos simples como enteros, flotantes y cadenas.
    -   `datos-compuestos.py`: Ejemplos de tipos de datos compuestos como listas, tuplas y diccionarios.
-   `variables/`: Carpeta que contiene ejemplos de cómo usar variables en Python.
    -   `variable.py`: Un script que muestra la declaración y uso de variables.

## Cómo usar

Cada archivo `.py` se puede ejecutar de forma independiente para ver los ejemplos en acción.
